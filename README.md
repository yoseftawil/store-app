# Simple Store App written in flask

### Setup with Pipenv:

- After cloning the repo:
```
pip3 install pipenv
pipenv shell
pipenv install
python3 app.py
```

- Site should be on localhost:5000

### Setup with Docker:

```
docker pull yoseftawil353/store-app
docker run -it -p 5000:5000 yoseftawil353/store-app
```
